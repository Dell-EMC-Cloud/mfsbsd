  - Make sure that the `ports` and `src` packages are selected during install

Note that at the end of installation, make sure to remove the ISO by following the instructions.

1. Add shared folder
poweroff

* Use shared folder from host server

. VirtualBox => Settings => Shared Folders => folder-add icon

. browse to the folder (let's presume the folder being selected is called "Shared"), select it then select "Auto-mount"

. boot FreeBSD guest OS and log in as root

mkdir /mnt/sf_(name)

sudo mount -t vboxvfs Downloads /mnt/shared

1. Make it mounted after reboot:
vi /boot/loader.conf
vboxvfs_load="YES"

vi /etc/fstab
Shared /sf_(name) vboxvfs rw 0 0

reboot
